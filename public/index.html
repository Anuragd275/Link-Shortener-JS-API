Link Shortener JS API

A REST API built with JavaScript and Express to short your links and also with the feature to connect your own domain to make your own personal URL Shortener

API Documentation

Welcome to our API documentation. This API can short your long URLs. Below you'll find details on available endpoints, request formats, and example responses.

Base URL

All API requests should be made to:

https://link-shortener-js-api.vercel.app/

Endpoints

1. Short a long URL

Pass your long URL to create a short url

  • URL: /short
  • Method: POST
  • Parameters:
    • long_url (string, required): Long URL that you want to short

Example Request

curl --location 'https://link-shortener-js-api.vercel.app/short' \ 
    --header 'Content-Type: application/json' \
    --data '{"long_url": "https://www.example.com"}'

Example Response

{
    "shortID": "cdb4d88d",
    "longURL": "https://www.example.com",
    "_id": "66c7855b76e819ccb7098116",
    "__v": 0
}

2. Retrieve a longURL

Get the value of long URL by its shortID

  • URL: /{shortID}
  • Method: GET
  • Parameters:
    • shortID (string, optional): The shortID of the longURL

Example Request

curl --location 'https://localhost:3000/{shortID}

Example Response

https://www.example.com

Error Handling

The API uses conventional HTTP response codes to indicate the success or failure of an API request. Codes in the 2xx range indicate success, codes in the 4xx range indicate an error that failed given the information provided (e.g., a required parameter was omitted), and codes in the 5xx range indicate an error with our servers.

Example Error Response

{
  "error": {
    "code": 400,
    "message": "Invalid request: Missing required field 'title'"
  }
}
